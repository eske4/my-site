Aalborg University - Project 3
Interactive Installation at Trekanten

In this project we created an interactive installation at Trekanten, a culture house, involving users in the design process. Through user research, personas, and scenarios, we made a solution for Trekanten. Employing Nielsen's heuristics and Gestalt theory, we refined the design and developed an augmented reality-based high-fidelity prototype. The prototype was showcased in a video created using SolidWorks and Photoshop Premiere.
